How to Get Started:
───────────────────
1. Get free Gemini API key: https://aistudio.google.com/app/apikey
2. Go to chrome://extensions → Enable Developer Mode
3. Click "Load unpacked" → Select the ai-reply folder
4. Click extension icon → "Open Settings" → Paste API key → Save
5. Go to messenger.com → Right-click on a message → "Generate Lease Reply"

Memory/Feedback System:
──────────────────────
How it works:
1. User highlights text and right-clicks
2. AI generates a reply using stored system prompt
3. Popup appears with reply + Copy button
4. User rates with 👍 or 👎 and optionally adds feedback
5. Feedback is appended to a markdown-like memory log
6. Next time an API call is made, memory log is appended to system prompt
7. AI learns and improves based on your preferences

Example memory entry:
  ## 2026-06-14 [GOOD] — great concise tone
  **Customer message:** Hey, interested in vending...
  **Reply snippet:** Great! We have several options...
